## 主要变化
1. 图片变大
2. 数据量变大
3. 模型深度变深
4. 使用Max-pool代替Average-pool,保存最重要的特征
5. 算力变强（它是首个证明 GPU 在大规模深度学习训练中巨大价值的里程碑式案例，其成功直接推动了 GPU 在 AI 训练领域的普及）
