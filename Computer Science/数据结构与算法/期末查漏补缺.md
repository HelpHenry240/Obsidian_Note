### 第一部分：现有复习框架的遗漏与补充（查漏补缺）

你的 `期末复习.md` 覆盖了80%的核心考点，但结合试卷，以下 **5个知识点** 在复习资料中较为薄弱或未提及，而考试中出现频率较高：

#### 1. 广义表（Generalized List）的深度运算
*   **遗漏点**：复习资料提到了 Head/Tail，但试卷（如填空题第21题）喜欢考 **嵌套运算**。
*   **真题考法**：`GetHead(GetTail(GetHead(GetTail(A))))`。
*   **补课**：
    *   **Tail 取尾**：结果**一定**要多加一层括号。例如 `A=(a, b, c)`，`Tail(A) = (b, c)`。
    *   **Head 取头**：结果**不加**最外层括号。例如 `Tail(A)=(b, c)`，对它取 `Head` 得到 `b`。
    *   **口诀**：Tail 剥皮留肉加皮，Head 剥皮取肉。

#### 2. 双向栈（共享栈）的代码实现
*   **遗漏点**：复习资料只提了概念，但试卷（算法题第2题）考了**完整的代码实现**（Init, Push, Pop）。
*   **核心逻辑**：
    *   一个数组 `V[n]`，两个栈底分别在 `0` 和 `n-1`。
    *   **栈满条件**：`top1 + 1 == top2`。
    *   **栈空条件**：`top1 == -1` (左栈) 或 `top2 == n` (右栈)。

#### 3. 森林与二叉树的转换及线索化画图
*   **遗漏点**：复习资料有转换口诀，但试卷（应用题31题）要求：
    1.  **根据先序+中序还原出“森林”**（注意是森林不是单一二叉树）。
    2.  **画出后序线索二叉树**（需画出虚线指针指向前驱后继）。
*   **补课**：
    *   森林的先序 = 对应二叉树的先序；森林的中序 = 对应二叉树的中序。
    *   先还原成二叉树，再断开右孩子连线还原成森林。

#### 4. 分块查找（索引顺序查找）
*   **遗漏点**：试卷选择题（2019年第13题）出现了“既能较快查找，又能适应动态变化”的选项。
*   **结论**：**分块查找**是这种场景的最佳选择。它结合了顺序查找（适应动态变化，链表实现）和二叉查找（索引表有序）的优点。

#### 5. 矩阵压缩存储的下标具体计算
*   **遗漏点**：试卷（选择题第8题）非常喜欢考 $n \times n$ 对称矩阵压缩到一维数组，求 $A[i][j]$ 的地址。
*   **陷阱**：必须看清是**行优先**还是**列优先**，以及下标是**从0开始**还是**从1开始**。
    *   公式不要死记，考场上用 `i=1, j=1` 或 `i=2, j=1` 这种小得数带进去验证最稳。

### 第二部分：试卷考点频率与分值分布（重点复习指南）

根据四套试卷的统计，分值分布非常固定，你的复习时间应按以下比例分配：

#### 1. 应用大题（占比 45% - 50%）—— **拿分主战场**
这部分题目是送分题，也是丢分题，**必须能够手工模拟每一步过程**。

*   **Top 1: 最小生成树 (Prim) + 最短路径 (Dijkstra) (必考)**
    *   **形式**：给一个图，让你画出 Prim 的选边顺序；填表写出 Dijkstra 的 `D[]` 数组变化。
    *   **复习要求**：Dijkstra 那个表（Visited, D[], Path[]）一定要会画，不能只写结果。
*   **Top 2: 哈希表 (Hash) (必考)**
    *   **形式**：给序列、表长、哈希函数。分别考 **线性探测法** 和 **拉链法**。
    *   **复习要求**：**ASL失败** 的计算是难点，一定要搞懂分母是哈希模数 $P$ 还是表长。
*   **Top 3: 排序算法模拟 (必考)**
    *   **形式**：**快速排序**（写第一趟Partition结果）、**堆排序**（画初始堆、筛选过程）、**归并排序**。
    *   **复习要求**：快排的“左右指针交替填坑”法必须熟练。
*   **Top 4: 二叉树/森林的遍历与还原**
    *   **形式**：给先序+中序，画树，写后序。或者给森林的遍历，画森林。

#### 2. 选择与填空（占比 35% - 40%）—— **概念细节**
*   **高频考点**：
    *   **时间复杂度**：$O(n^2)$，$O(n \log n)$ 的代码判断。
    *   **数据结构三要素**：逻辑结构、存储结构、运算。
    *   **二叉树性质**：$n_0 = n_2 + 1$ （已知叶子求总数，或已知总数求叶子）。
    *   **完全二叉树**：节点编号 $i$ 的左孩子 $2i$。
    *   **栈与队列**：出栈序列合法性判断（卡特兰数逻辑）、循环队列判满 `(rear+1)%n == front`。
    *   **串**：KMP算法的 `next` 数组值（手算）。

#### 3. 算法设计（占比 10% - 15%）—— **代码题**
试卷中的代码题难度并不像考研那么大，主要集中在：
*   **链表操作**：拆分链表（正数一个表，负数一个表）、删除重复节点、归并链表。
*   **二叉树递归**：统计度为1的节点个数、交换左右子树、求高度。
*   **特殊结构**：双向栈的操作（见上文）。

### 第三部分：冲刺阶段复习建议

**1. 必背公式与结论（写在草稿纸上备用）**
*   循环队列长度：`(rear - front + maxSize) % maxSize`
*   二叉树叶子公式：$n_0 = n_2 + 1$
*   ASL计算：
    *   成功 = (每项比较次数之和) / 元素个数
    *   失败 = (每个位置到空位的比较次数之和) / 模数 $P$
*   排序复杂度表格：特别是**堆排序**、**归并排序**、**快排**的时间/空间/稳定性。

**2. 重点突击题型（按优先级排序）**
1.  **[画图]** 练习 Dijkstra 算法的列表推导过程（不要只画图，要写数组变化）。
2.  **[计算]** 练习哈希表线性探测法的 ASL 失败计算（极易出错）。
3.  **[画图]** 练习由数组构建 **大顶堆** 的过程（从下往上调整）。
4.  **[代码]** 背诵链表节点的定义，熟练手写 `p = p->next` 遍历和插入删除逻辑。

**3. 避坑指南**
*   **KMP Next 数组**：西电试卷中的 Next 数组通常**从 1 开始**（即第1个字符的next是0），注意看题目是否暗示数组下标从0开始（那样next就是-1）。
*   **Prim vs Kruskal**：题目常指定用 Prim（普里姆）算法，别做成 Kruskal（克鲁斯卡尔）。Prim 是从点出发找最近的边，Kruskal 是全图找最小边防环。

按照这个补充框架，重点练习 **应用大题的手工模拟**，你的期末成绩应该会很稳。
